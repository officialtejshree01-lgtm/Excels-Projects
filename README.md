# 📊 Excel Projects Portfolio

## 📌 Project Overview
This repository contains a collection of **Excel-based Dashboard Projects** developed across multiple business domains.  
Each project demonstrates the use of **data analysis, visualization, and dashboard design techniques** to solve real-world business problems and support data-driven decision-making.

---

## 🧾 Projects Included

### 🏦 1. Bank Customer Loan Tracker  
**Domain:** Banking & Financial Services  

**Overview:**  
An Excel dashboard designed to track customer loans, monitor repayment status, and identify overdue or defaulted accounts.

**Key Features:**
- Loan tracking and repayment monitoring  
- Overdue & default loan identification  
- Loan type distribution analysis  
- Outstanding balance insights  

---

### 😊 2. Customer Feedback Analysis  
**Domain:** Customer Experience / Service Quality  

**Overview:**  
An interactive Excel dashboard to analyze customer feedback, measure satisfaction levels, and identify service improvement areas.

**Key Features:**
- Customer rating analysis  
- Feedback category distribution  
- Resolved vs pending case tracking  
- Feedback trend analysis  

---

### 📦 3. Retail Store Inventory Management  
**Domain:** Retail / Inventory & Supply Chain  

**Overview:**  
An Excel dashboard focused on managing inventory levels, tracking stock movement, and improving supply chain efficiency.

**Key Features:**
- Inventory tracking and stock monitoring  
- Low stock and out-of-stock alerts  
- Category-wise inventory analysis  
- Inventory movement trends
