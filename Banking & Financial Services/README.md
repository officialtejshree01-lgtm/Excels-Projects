# 📊 Bank Customer Loan Tracker (Excel Dashboard)

## 📌 Project Overview
This project is an **Excel-based Loan Tracking Dashboard** developed for a **Banking & Financial Services company**.  
It helps track loan applications, analyze customer credit profiles, and monitor approval trends to support better decision-making.

---

## 🎯 Objectives
- Track **loan applications and approval status**
- Analyze **credit score impact on loan approval**
- Identify **high-risk vs low-risk customers**
- Monitor **region-wise loan performance**
- Build an **interactive dashboard for insights**

---

## 🛠️ Tools & Skills Used
- Microsoft Excel  
- Data Cleaning & Transformation  
- Pivot Tables & Charts  
- Conditional Formatting  
- Excel Formulas (IF, COUNTIF, SUM, etc.)  
- Dashboard Design & KPI Metrics  

---

## 📂 Dataset Description
The project includes multiple datasets:
- **Customers** → Customer personal details  
- **Loans** → Loan application records  
- **Credit History** → Credit score & repayment behavior  
- **Regions** → Regional classification  

---

## 🔄 Data Processing Steps
- Removed duplicates using **Customer_ID**
- Cleaned and standardized text fields
- Handled missing values (replaced with "Unknown")
- Converted numerical columns into proper format
- Merged datasets using **Customer_ID**
- Created calculated fields:
  - Loan Approval Flag  
  - Risk Category  
  - Loan-to-Income Ratio  

---

## 📊 Dashboard Features
- 📌 **Loan Approval Rate (%)**
- 📌 **Credit Score Analysis**
- 📌 **Region-wise Loan Distribution**
- 📌 **Risk Segmentation (High vs Normal)**
- 📌 **Interactive Filters (Slicers)**

---

## 📈 Key Insights
- Customers with higher credit scores have better approval chances  
- Certain regions show higher loan approval rates  
- High Loan-to-Income ratio indicates higher risk  
- Credit history significantly impacts loan decisions
