# 📊 Customer Feedback Analysis (Excel Project)

## 📌 Project Overview  
This project focuses on analyzing customer feedback data for a service-based company using Microsoft Excel. The objective is to extract meaningful insights that help improve customer satisfaction and enhance business performance.

---

## 🎯 Objectives  
- Identify customer satisfaction trends  
- Analyze feedback ratings  
- Detect areas of improvement  
- Support data-driven decision making  

---

## 🛠️ Tools & Techniques Used  
- Microsoft Excel  
- Pivot Tables  
- Charts (Bar, Pie, Line, Histogram)  
- Conditional Formatting  
- What-If Analysis  
- Data Cleaning & Transformation  

---

## 📂 Dataset  
The dataset includes the following fields:  
- Customer Name / ID  
- Feedback Rating  
- Service Category  
- Comments  
- Date  

---

## 📈 Key Insights  
- Identified customers who gave the lowest ratings  
- Analyzed trends in customer satisfaction over time  
- Highlighted major service issues  
- Compared performance across different categories
